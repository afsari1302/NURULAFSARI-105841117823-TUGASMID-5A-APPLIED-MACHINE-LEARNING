Repository ini berisi seluruh proses pengerjaan Tugas Besar Applied Machine Learning dengan studi kasus prediksi kategori jumlah siswa SD berdasarkan jumlah siswa dari kelas 1 hingga kelas 6 di setiap provinsi di Indonesia. Proyek ini disusun oleh Nurul Afsari (105841117823) sebagai bagian dari pemenuhan tugas mata kuliah Applied Machine Learning kelas 5A AI-A. Seluruh file yang terdapat di dalam repositori ini mencakup dokumen perancangan proyek, laporan lengkap, notebook pemrograman, dataset yang digunakan, dan model Machine Learning yang telah dilatih.

Proyek dimulai dari proses pemahaman data dengan memanfaatkan dataset jumlah siswa laki-laki per tingkat SD berdasarkan provinsi. Dataset ini kemudian dibersihkan melalui beberapa tahap preprocessing, seperti penghapusan karakter non-angka, konversi nilai menjadi numerik, dan perbaikan struktur tabel agar siap digunakan dalam analisis. Setelah data bersih, dibuatlah fitur baru berupa total siswa per provinsi yang kemudian dikategorikan menjadi tiga kelas, yaitu Rendah, Sedang, dan Tinggi, dengan menggunakan pembagian kuantil. Kategori inilah yang menjadi target dalam proses klasifikasi.

Setelah data siap, proyek dilanjutkan dengan langkah pembagian data menjadi train dan test menggunakan proporsi 70% untuk pelatihan dan 30% untuk pengujian. Model yang digunakan adalah algoritma Random Forest, yang dirangkai dalam sebuah pipeline bersama StandardScaler untuk memastikan proses preprocessing dan training berjalan konsisten. Model kemudian dilatih menggunakan data train dan dievaluasi menggunakan confusion matrix serta classification report. Hasil evaluasi menunjukkan bahwa model mampu melakukan prediksi dengan akurasi sempurna pada dataset uji, sehingga dianggap layak untuk digunakan dalam tahap deployment.

Model yang telah dilatih kemudian disimpan ke dalam file .pkl menggunakan joblib sehingga dapat dimuat ulang tanpa harus melatih ulang dari awal. Selanjutnya, model diimplementasikan dalam sebuah aplikasi berbasis web menggunakan Gradio. Aplikasi ini memungkinkan pengguna memasukkan jumlah siswa kelas 1 hingga kelas 6, dan aplikasi akan mengembalikan kategori jumlah siswa SD (Rendah, Sedang, atau Tinggi) beserta probabilitas prediksinya. Aplikasi ini berjalan secara lokal melalui browser dan dirancang agar mudah digunakan oleh siapa pun tanpa perlu memahami kode program.

Seluruh berkas yang terkait proyek ini telah diunggah ke dalam repository GitHub agar dapat diakses dan diperiksa secara terbuka, sesuai instruksi tugas. Repository ini diharapkan dapat memberikan gambaran lengkap mengenai alur kerja pengembangan sebuah model Machine Learning mulai dari tahap pengolahan dataset, pembuatan fitur, pelatihan model, hingga deployment aplikasi prediksi. Dokumentasi ini juga ditujukan agar pengguna atau penguji dapat memahami maksud, tujuan, metode, serta hasil akhir dari proyek yang telah diselesaikan.
